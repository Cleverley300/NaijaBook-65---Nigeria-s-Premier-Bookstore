# NaijaBook-65---Nigeria-s-Premier-Bookstore
NaijaBook@65 is a front-end bookstore web project built with HTML5, CSS3 and JS, inspired by Nigeria’s literary culture. It features a modern, responsive layout with a sticky header, hero section, and grid-based book catalog.  Key Highlights   • Fully responsive and mobile-optimized design • Accessible navigation  Skills: HTML5 · CSS3 · JS ·
